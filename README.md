# 🧭 Learning Roadmap

## 🔹 Phase 1: Python Foundations

**Focus:** Logic building + core syntax

Topics covered:

* Variables, data types
* Loops & conditionals
* Functions
* Lists, dictionaries
* Basic problem solving

📌 Outcome:

* Comfortable writing Python scripts
* Able to solve logical problems

---

## 🔹 Phase 2: Data Visualization (Matplotlib)

**Goal:** Understand how to represent data visually

### 📊 Plots Covered:

* Line Plot
* Bar Chart
* Horizontal Bar
* Pie Chart
* Stack Plot
* Histogram
* Scatter Plot
* Subplots (Dashboard)

### 🧠 Key Learnings:

* Choosing the right plot for the data
* Labeling (x, y, title, legend)
* Grid & styling
* Avoiding clutter and confusion
* Understanding data relationships (especially scatter plots)

📌 Outcome:

* Can build clean and meaningful visualizations
* Able to create dashboard-style plots

---

## 🔹 Phase 3: HTML Basics (for Web Scraping)

**Goal:** Understand structure of web pages

Topics:

* Tags (`div`, `p`, `h1`, `a`, `img`)
* Lists & tables
* Classes & IDs
* Nested structure

📌 Outcome:

* Can inspect and understand webpage structure
* Ready for scraping real data

---

## 🔹 Phase 4: Web Scraping

### ⚙️ Tools Used:

* `requests` → Fetch HTML
* `BeautifulSoup` → Extract data

### 🧪 Skills Practiced:

* Sending HTTP requests
* Handling headers & status codes
* Parsing HTML
* Extracting:

  * Titles
  * Prices
  * Ratings
  * Stock availability

### 🧠 Key Learnings:

* Handling missing elements (`NoneType errors`)
* Cleaning messy text (`\n`, symbols, spaces)
* Extracting structured data from unstructured HTML

📌 Outcome:

* Built a working scraper
* Extracted multi-page data
* Understood real-world scraping issues

---

## 🔹 Phase 5: Data Cleaning & Structuring

### Tools:

* `pandas`

### Tasks:

* Convert scraped data → DataFrame
* Clean values (price, text, etc.)
* Handle missing values
* Prepare dataset for analysis

📌 Outcome:

* Can transform raw data into structured format

---

## 🔹 Phase 6: SQL Fundamentals

### 🧪 Skills Covered:

* Creating database & tables
* Inserting data
* Updating records

* Altering table structure
* Filtering & sorting data

### 🧠 Key Learnings:

* Data storage & querying
