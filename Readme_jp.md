## 🔹 フェーズ4: ウェブスクレイピング

### ⚙️ 使用ツール：

* `requests` → HTMLを取得
* `BeautifulSoup` → データを抽出

### 🧪 練習スキル：

* HTTPリクエスト送信
* ヘッダーとステータスコード処理
* HTMLパーシング
* 抽出：

  * タイトル
  * 価格
  * 評価
  * 在庫状況

### 🧠 主な学習:

* 欠落要素の処理（`NoneType errors`）
* ノイズテキスト（`\n`、記号、スペース）クリーニング
* 非構造化HTMLから構造化データを抽出

📌 成果：

* 動作するスクレイパーを構築
* マルチページデータを抽出
* 実世界のスクレイピング問題を理解
